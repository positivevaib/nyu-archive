Assignment 3 - Banker

The code is implemented in Java.

The source code is in the Banker.java file.


Class descriptions:

- Banker class is the main class and contains the two resource allocating algorithms along with other subroutines to help with resource allocation.

- Task class is used to store and track the state and activities information for different tasks.


Running the program:

- A simple way to run the program is to use the command line, compiling with javac and running the executable with java. The input is read through files, so the filename needs to be provided as a command line argument.
	javac Banker.java
	java Banker input-x
